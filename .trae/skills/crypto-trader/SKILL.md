# AI 加密货币合约交易系统

## 项目概述

一个全自动化的 Binance U本位合约量化交易系统。核心策略是**快捞**：每1分钟检测500+币种，15分钟涨>7.3%做多/跌>7.3%做空，小仓追涨杀跌 + 固定5%价格回撤锁仓止盈。同时每10分钟检查资金费率，每笔交易记录滑点成本。

**核心逻辑**：监测(1min) → 发现异常涨跌(>7.3%) → K线趋势过滤 → 小仓开仓(10%余额,5x杠杆) → 5秒持仓监控(固定5%回撤锁仓) → 滑点追踪 → 资金费率追踪(10min) → 1h/3h/6h/12h/24h微信汇总

## 快捞策略参数

| 参数 | 值 | 说明 |
|------|-----|------|
| 杠杆 | 5x | 每仓5倍杠杆 |
| 仓位 | 10% 余额 | 每仓占总额10% |
| 触发 | >7.3% | 15分钟涨/跌超7.3% |
| 冷却 | 10分钟 | 止损后冷却（固定时间，不提前解除） |
| 同时持仓 | ≤3个 | 最多3个快捞单 |
| 锁仓 | 固定5% | 入场价回撤5%止盈 |
| 持仓监控 | 每5秒 | 检查已有快捞仓位的止损/锁仓 |

## 3个并行循环

| 循环 | 频率 | 职责 |
|------|------|------|
| **持仓监控循环** | 每5秒 | 检查已有快捞仓位的止损/锁仓 |
| **快捞监测循环** | 每1分钟 | 扫描500+币种，15min涨>7.3%做多/跌>7.3%做空 |
| **汇总报表循环** | 每分钟检查 | 到点发1h/3h/6h/12h/24h微信汇总 |

## 新增功能

- **入场模式记录** — 记录每笔开仓的入场模式（trend/volatility_override/sideways）
- **滑点追踪** — 每笔交易记录滑点成本，支持性能分析
- **资金费率追踪** — 每10分钟检查一次持仓币种资金费率
- **实时价格检查** — 测试网模式下使用实盘真实价格进行判断
- **5秒持仓监控** — 新增高频率持仓监控循环，及时处理止损/锁仓

## 运行方式

```bash
# 生产模式（3个循环并行）
uv run python scheduler.py forever

# 前端看板
uv run python run_dashboard.py

# 查余额
uv run python check_balance.py
```

## 配置说明

`config.py` 包含所有敏感信息：

- `TESTNET_API_KEY / TESTNET_SECRET` — 测试网密钥
- `MAINNET_API_KEY / MAINNET_SECRET` — 实盘密钥（⚠️ 注意保护）
- `DEEPSEEK_API_KEY` — AI 分析的 API Key
- `PUSHPLUS_TOKEN` — 微信通知 Token
- `USE_TESTNET` — `True` 测试网 / `False` 实盘
