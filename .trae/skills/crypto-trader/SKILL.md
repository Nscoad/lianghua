# AI 加密货币合约交易系统

## 项目概述

一个全自动化的 Binance U本位合约量化交易系统。核心策略是**快捞**：每2分钟检测500+币种，15分钟涨>6%做多/跌>6%做空，小仓追涨杀跌 + K线动态锁仓止盈。同时每30分钟采集币安广场和X.com动态，AI摘要后发微信。

**核心逻辑**：监测(2min) → 发现异常涨跌(>6%) → K线趋势过滤 → 小仓开仓(10%余额,5x杠杆) → 动态锁仓(K线波动算回撤) → 趋势采集(30min广场+X→AI摘要→微信) → 1h/3h/6h/12h/24h微信汇总

## 快捞策略参数

| 参数 | 值 | 说明 |
|------|-----|------|
| 杠杆 | 5x | 每仓5倍杠杆 |
| 仓位 | 10% 余额 | 每仓占总额10% |
| 触发 | >6% | 15分钟涨/跌超6% |
| 同时持仓 | ≤3个 | 最多3个快捞单 |
| 锁仓 | 动态 | ≥30%利润用K线波动算回撤 |

## 3个并行循环

| 循环 | 频率 | 职责 |
|------|------|------|
| **快捞循环** | 每2分钟 | 监测500+币种，15min涨>6%做多/跌>6%做空 |
| **趋势采集循环** | 每30分钟 | 广场+X → AI摘要 → 微信通知 |
| **汇总报表循环** | 每分钟检查 | 到点发1h/3h/6h/12h/24h微信汇总 |

## 运行方式

```bash
# 生产模式（3个循环并行）
uv run python scheduler.py forever

# 前端看板
uv run python run_dashboard.py

# 查余额
uv run python check_balance.py
```

## 微信汇总报表

| 周期 | 发送时间 | 内容 |
|------|---------|------|
| 趋势摘要 | 每30分钟 | 广场+X最新动态 AI 摘要 |
| 1小时 | 每个整点 | 盈亏、多空比 |
| 3小时 | 每3小时整点 | 盈亏、多空比 |
| 6小时 | 每6小时整点 | 盈亏、多空比、原因分布 |
| 12小时 | 每12小时整点 | 盈亏、多空比、币种明细 |
| 24小时 | 每天00:05 | 完整汇总 + AI复盘分析 |

## 配置说明

`config.py` 包含所有敏感信息：

- `TESTNET_API_KEY / TESTNET_SECRET` — 测试网密钥
- `MAINNET_API_KEY / MAINNET_SECRET` — 实盘密钥（⚠️ 注意保护）
- `DEEPSEEK_API_KEY` — AI 分析的 API Key
- `PUSHPLUS_TOKEN` — 微信通知 Token
- `USE_TESTNET` — `True` 测试网 / `False` 实盘
